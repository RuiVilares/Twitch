Exercise: Twitch Client

Write a web application to search for Twitch streams. The application should also allow the user to select any of the results and start watching the respective stream.

Requirements

You must write instructions on how to run the application.

Functional Requirements

  • Allow the user to search for streams;
  • Allow the user to choose the number of results when performing a search;
    • Remember this value even when the browser is closed)
  • Allow the user to open a stream from the search results;
    • Keep the the viewers count updated;

Extras

  • Application:
    • Cross Browser compatibility with modern Web Browsers;
    • Usage feedback (e.g. loading state, error handling);
    • Performance aware (e.g. don't overload the Twitch service);
    • Incremental Search;
    • Responsive UI;
    • Routing;
  • Development:
    • Use some kind of style sheet;
    • Use build tools (e.g. grunt, gulp, webpack)
    • Different environments for development and production
    • Unit tests
    • Functional tests
    • Clean Code
  • Amaze us ;)
